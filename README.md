# hyprland-plugins

A collection of custom plugins for Hyprland window manager.

## Plugins

### Mouse Gestures
Execute commands using mouse gestures with fuzzy pattern matching based on the easystroke algorithm.

[Demo Video](mouse-gestures/mouse-gestures.mp4) | [Documentation](mouse-gestures/README.md)

### Window Actions
Add customizable action buttons to window decorations with support for custom colors, icons, and state-dependent behavior.

[Demo Video](window-actions/window-actions.mp4) | [Documentation](window-actions/README.md)

### Workspace Overview
Visual workspace overview similar to GNOME's Activities view with split-screen layout and click-to-switch functionality.

[Demo Video](workspace-overview/workspace-overview.mp4) | [Documentation](workspace-overview/README.md)
